> 版本 v1.0 · 2026年6月

---

## 一、品牌调性

**一家食光**（Yī Jiā Shí Guāng）是一款中式风格的餐饮点单应用，融合传统水墨插画美学与现代移动端交互体验。

**关键词：** 雅致 · 温润 · 东方 · 沉静 · 亲切

**视觉风格：**
- 以**鸦青色 #285B4E** 为核心品牌色，沉稳克制，富有东方气韵
- 背景以白色与极淡青绿为主，营造清爽通透的阅读体验
- 插图采用工笔/水墨风格，搭配手绘质感装饰元素
- 字体混合楷体装饰标题与无衬线正文，中英文混排和谐

## 七、组件规范

### 7.1 按钮 Button

#### Primary Button（主要按钮）
```
背景色：--color-primary (#285B4E)
文字色：#FFFFFF
圆角：--radius-full (胶囊) 或 --radius-md (8px, 方形场景)
字号：15px / 500
高度：44px（标准）/ 36px（紧凑）
按压态：背景色变为 --color-primary-dark (#1C4238)
禁用态：背景色 #B8D6CB，文字 #FFFFFF，不可点击
```

```css
.btn-primary {
  background: #285B4E;
  color: #fff;
  font-size: 15px;
  font-weight: 500;
  height: 44px;
  padding: 0 24px;
  border-radius: 9999px;
  border: none;
}
.btn-primary:active { background: #1C4238; }
.btn-primary:disabled { background: #B8D6CB; }
```

#### Secondary Button（次要按钮）
```
背景色：--color-primary-50 (#EBF5F0)
文字色：--color-primary (#285B4E)
边框：1px solid --color-primary-100 (#D4EAE3)
其余同主要按钮
```

#### Outline Button（描边按钮）
```
背景色：transparent
文字色：--color-primary (#285B4E)
边框：1.5px solid --color-primary (#285B4E)
```

#### Ghost Button（幽灵按钮）
```
背景色：transparent
文字色：--color-text-secondary (#555555)
边框：1px solid --color-border (#E0EEEA)
```

## 十、交互规范

### 10.1 点击反馈
- 按钮按压：`scale(0.97)` + 背景色加深，持续 100ms
- 列表项点击：背景色过渡到 `--color-primary-25`，150ms
- 图标点击：`opacity: 0.7`，100ms

### 10.2 页面转场
- Push（前进）：新页面从右侧滑入，300ms ease-out
- Pop（返回）：当前页面向右推出，300ms ease-in
- 模态弹出：从底部上滑，背景渐显遮罩，250ms

### 10.3 加载状态
- 骨架屏：背景 `#EBF5F0`，高光 `#F5FAF7`，shimmer 动画
- 刷新：下拉释放时品牌色 loading 圆圈
- 按钮 loading：内嵌旋转圆圈，白色，禁止点击

### 10.4 Toast / 提示
- 背景：`rgba(28, 66, 56, 0.90)`（深鸦青透明）
- 文字：#FFFFFF，14px
- 圆角：8px，内边距 10px 16px
- 出现位置：屏幕 60% 高度处，居中
- 持续时间：2s，淡入淡出各 200ms

## 十一、设计原则

1. **留白呼吸**：不要让内容过分拥挤，用充足的间距营造东方美学的从容感
2. **色彩克制**：主色 #285B4E 作为少量点睛色，大面积底色保持素净
3. **插画优先**：遇到纯文字区块，尽量补充传统风格插画或纹样装饰
4. **文案亲切**：使用"小主"等品牌化称呼，文案简洁不生硬
5. **层次分明**：通过背景色深浅（25/50/100）而非阴影过多来区分层次
6. **品质感**：圆角不要过大（多用 8px），金色装饰适度点缀
